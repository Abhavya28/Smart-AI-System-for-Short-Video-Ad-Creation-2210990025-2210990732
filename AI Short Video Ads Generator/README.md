# AI Short Video Ads Generator 

AI Short Video Ads Generator is an AI powered short video ads generation platform that enables users to create high quality advertising creatives using artificial intelligence. 

Users can upload a model image and a product image, and the system generates a realistic, seamlessly blended image along with a short promotional video.

The platform is built to simplify and automate ad creation for ecommerce, social media marketing, and digital campaigns, allowing users to generate professional grade marketing content without requiring any editing or design skills.

## Setup Instructions

```bash
# Clone repository
git clone <repo-url>

# Install client dependencies
cd client
npm install
npm run dev

# Install server dependencies
cd server
npm install
npm start
```
